# Spotify Data Mining Project

## Quy trình thực hiện (CRISP-DM)
1. **Integration**: Kết hợp dữ liệu gốc và Spotify API.
2. **Cleaning**: Làm sạch và chuẩn hóa dữ liệu.
3. **EDA**: Phân tích khám phá và trực quan hóa.
4. **Feature Engineering**: Biến đổi đặc trưng.
5. **Modeling**: Triển khai các thuật toán khai phá.
